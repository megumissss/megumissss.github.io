# 欢迎来到我的博客！

这是我博客的第一篇文章，用来测试博客的功能和展示一些基本特性。

## 博客功能

这个博客基于Hugo静态站点生成器，使用PaperMod主题，包含以下功能：

- ✅ 响应式设计
- ✅ 暗色/亮色模式切换
- ✅ 搜索功能
- ✅ 分类和标签系统
- ✅ 评论系统（待配置）
- ✅ 相册和项目展示

## 技术栈

- **静态站点生成器**: Hugo
- **主题**: PaperMod
- **部署**: GitHub Pages
- **版本控制**: Git
